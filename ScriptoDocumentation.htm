What is Scripto Algorithm?

Scripto is a permission-based metadata extraction engine. It activates only when a page explicitly defines:

const url_Id = "UNIQUE_PAGE_ID";

After a 6-second delay, it extracts:

  • Page URL
  • Title
  • Description
  • Logo (OG Image / Favicon / Fallback Scan)

Scripto Execution Flow

Page Load ↓ Wait 6 Seconds ↓ Check Session ↓ Extract Metadata ↓ Send to Collector ↓ Store in IndexedDB ↓ Display in Dashboard

ANH Hash Algorithm (Public Overview)

The ANH Hash is a custom password transformation system.

  • Convert each character to ASCII
  • Inline numeric concatenation
  • After every 6 digits insert rotating token
  • Store only transformed output

Example (Partial):

Password → ASCII → 8011597... Insert → 801159anh97...

Full transformation rules remain private for security integrity.

Database Architecture

Database: ANH_DB Object Stores: - users - analytics - sessions - guestVisits

All storage is performed locally via IndexedDB. No server transmission occurs.

Security Model

  • Session-based validation (30 min expiry)
  • Guest access limit (3 pages/day)
  • IndexedDB structured isolation
  • No cloud storage
  • No third-party trackers
  • Explicit script permission required
